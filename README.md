## Features

- Convert Audio (mic, .wav, .mp3, .AIFF, .AIFF-C, FLAC) to Text (.txt, .srt, .vtt)

## Version 0.0

This initial release focuses on audio transcription.

### Supported Formats

- **Input Formats**: mic, .wav, .mp3, .AIFF, .AIFF-C, FLAC
- **Output Formats**: .txt, .srt, .vtt
